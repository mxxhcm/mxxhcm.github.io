## 安装
安装git
~$:sudo apt install git
安装nodejs
~$:sudo apt install nodejs
安装npm
~$:sudo apt install npm
安装hexo 
~$:sudo npm install -g hexo-cli

## 配置
创建文件夹
~$:git clone your repo
或者直接
~$:hexo init your repo
安装依赖包
~$:npm install
